-=HomeTask=-
авдання:
Ваше завдання полягає у створенні простого додатку ToDo List з використанням JDBC для зберігання та управління списком
завдань. Нижче наведена покрокова інструкція, яку ви повинні дотримуватися для виконання завдання:

Крок 1: Підготовка бази даних

    Створіть базу даних todolist в MariaDB.
    Створіть таблицю tasks з наступними полями:
        id (ціле число, автоінкрементне),
        description (рядок, обов'язкове),
        status (BOOLEAN або INT для позначення статусу завдання).

Крок 2: Налаштування проєкту

    Створіть новий maven проєкт Java в Itellij IDEA.
    Додайте необхідну бібліотеку для з'єднання з базою даних через JDBC.

Крок 3: Реалізація функцій ToDo List

    Створіть клас Task для представлення завдання з полями id, description та status.
    Створіть клас TaskDao з методами:
        addTask(Task task): для додавання нового завдання.
        getAllTasks(): для отримання списку всіх завдань.
        updateTaskStatus(int taskId, boolean status): для оновлення статусу завдання за id.
        deleteTask(int taskId): для видалення завдання за id.

Крок 4: Створення користувацького інтерфейсу

    Створіть консольний інтерфейс для користувача з наступними опціями:
        Додати завдання.
        Переглянути список завдань.
        Позначити завдання як виконане.
        Видалити завдання.

Крок 5: Завершення проєкту

    Протестируйте свій додаток, переконайтеся, що всі функції працюють правильно та взаємодіють з базою даних через JDBC.

Оцінювання:
Оцінка буде здійснюватися з урахуванням наступних критеріїв:

    Правильність та ефективність реалізації функцій ToDo List.
    Відповідність коду стандартам програмування та організація коду.
    Використання JDBC для роботи з базою даних.
    Функціональність та зручність користувацького інтерфейсу.
    Організація звіту та демонстрація роботи додатку.
